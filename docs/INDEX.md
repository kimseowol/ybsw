# 3축 추적 매트릭스

기획(GD) ↔ PRD ↔ 코드의 연결을 추적한다. PRD 생성·상태 변경 시 함께 갱신한다.

연결 상태 라벨: `미구현`(PRD만 있음) / `코드 미확인`(코드가 있는지 확인 안 됨) / `기획 확인 필요`(기획 문서와의 정합 확인 필요). 라벨 없는 빈칸을 남기지 않는다.
